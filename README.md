# Simple Todo App

This app helps you to keeps track of your everyday responsibilities. It includes being able to add any tasks you may encounter along with dates and being able to mark them as you go along.

## Functionality

- Users can add ToDo items
- Mark off any finished tasks
- Add due dates

## Technology

- JavaScript (Vanilla, ES6 Modules)
- DOM Manipulation
- UUid Libraries
